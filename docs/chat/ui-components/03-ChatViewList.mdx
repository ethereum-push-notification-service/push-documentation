- **ChatViewList**
    
    ### Want something like an announcement board ? Import ChatViewList, add the chatid of the group and you can do it just in few lines of code!
    
    - Inputs
        - chatId - required
        - limit - optional (default value 10)
    - Example
        
        - ![Example Image](https://github.com/KlausMikhaelson/KlausMikhaelson/assets/100528412/0b33006c-b931-453f-b7f1-c26894a82516)
